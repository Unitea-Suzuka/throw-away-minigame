# ごみすてゲーム(throw-away-minigame)
## 概要・目的
この作品は、幼児向けにデザインされた社会問題をテーマにしたミニゲーム集です。<br>
⽂字を⼀切使⽤せず、イラストや直感的な操作で遊べるため、まだ⽂字が読めない⼦どもでも楽しめます。<br>
⽬的は、幼児期から世界の課題を知るきっかけを作り、将来的に「持続可能な社会」への意識を育むことです。<br>

## 開発環境
- Unity6000.2.6F2
- Visual Studio 2022
